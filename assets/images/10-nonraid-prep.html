NonRAID — Disk Prep
Step 5 of 8 — partition, format, and start the array
--:--:--
Setup Mode
NonRAID
Parity Drives — GPT type fd00
⚠ Destructive — run as root
/dev/sdb — parity
sgdisk --zap-all /dev/sdb
sgdisk -n 1:0:0 -t 1:fd00 /dev/sdb
Data Drives — GPT type 8300, then mkfs.xfs
/dev/sdc — data
sgdisk --zap-all /dev/sdc
sgdisk -n 1:0:0 -t 1:8300 /dev/sdc
mkfs.xfs -f /dev/sdc1
/dev/sdd — data
sgdisk --zap-all /dev/sdd
sgdisk -n 1:0:0 -t 1:8300 /dev/sdd
mkfs.xfs -f /dev/sdd1
/dev/sde — data
sgdisk --zap-all /dev/sde
sgdisk -n 1:0:0 -t 1:8300 /dev/sde
mkfs.xfs -f /dev/sde1
Start the Array — interactive (run in terminal)
nmdctl create
nmdctl create is interactive — it prompts to confirm each drive assignment.
After completing the wizard, run nmdctl start then nmdctl mount to bring the array online.
